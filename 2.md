Question
Mitchell has invented a machine that outputs the most frequently occurring characters in a string that lie in the range [L, R]. The machine accepts a series of characters and asks the user to input two numbers, L and R. The machine outputs the characters for all the pairs of [L, R] values the user provides.

Write an algorithm to help Mitchell find the output for all the inputs he provides.

Input
The first line of the input consists of a string str, representing the series of characters that are fed to the machine initially.
The second line consists of an integer P, representing the number of pairs provided by the user.
The next P lines consist of two space-separated integers - L and R, representing the values of the lower and upper range for the pairs [L, R], respectively.

Output
Print P characters representing the most frequently occurring characters for all the pairs of inputs Mitchell provides. If more than one character in a particular input has the same frequency, print the character that is least in the lexicographic order.

Constraints
1 ≤ P ≤ 10⁵
1 ≤ L ≤ R ≤ len; where len is size of str

Note
All the characters that are fed to the machine are lowercase English letters.

Example
Input:
cabcac
3
1 6
1 3
2 5

Output:
caa

Explanation:
String is "cabcac":
For range [1,6], 'c' occurs thrice, and is the most frequent.
For range [1,3], 'a', 'b', 'c', all occur once but 'a' is lexicographically smallest.
For range [2,5], 'a' occurs twice, and is the most frequent.
So, the final output is "caa."
